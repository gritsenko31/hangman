## ğŸ® How to Play

### Objective
Guess the hidden word by selecting letters one at a time. You have 6 wrong guesses before the game is over.

### Controls

**Desktop:**
- **Click Letter Buttons** - Guess a letter
- **Keyboard Keys** - Type letters directly (A-Z)
- **New Game Button** - Start with a new word

**Mobile:**
- **Tap Letter Buttons** - Select your guess
- **Tap New Game** - Get a fresh word

### Game Rules
- Each correct letter reveals all instances of that letter in the word
- Each wrong letter adds a body part to the hangman (6 parts total)
- **Win:** Reveal all letters before the hangman is complete
- **Lose:** Complete the hangman drawing (6 wrong guesses)
- Used letters are disabled to prevent duplicate guesses

### Hangman Stages
1. Head
2. Body
3. Left Arm
4. Right Arm
5. Left Leg
6. Right Leg (Game Over)
